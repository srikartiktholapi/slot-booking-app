# Slot Booking App

A simple slot booking web application using Python and MySQL.

## Tech Stack
- Frontend: React
- Backend: Python, Django REST Framework
- Database: MySQL

## Features
- View available slots
- Book a slot
- See booked slots

## Run Locally

**Backend**
```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

**Frontend**
```bash
cd frontend
npm install
npm start
```
